# 02 변수와 자료형

## 02-1 컴퓨터에서 데이터 표현하기

컴퓨터는 0과 1로만 데이터를 저장한다.

- **bit**(비트): 컴퓨터가 표현하는 데이터의 최소 단위. 2진수 하나의 값을 저장할 수 있는 메모리 크기.
- **byte**(바이트): 1byte=8bit

### 부호 있는 수 표현하기

- MSB(Most Significant Bit): 가장 중요한 비트, 부호를 나타냄. 0이면 양수 1이면 음수
- 1의 보수: 0과 1의 값을 반전시킨 값
- 2의 보수(음수): 1의 보수 + 1 의 값
