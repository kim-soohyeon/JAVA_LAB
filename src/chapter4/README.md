# 04 제어 흐름 이해하기

## 04-1 조건문
---
### 조건문이란?

주어진 조건에 따라 다른 수행문이 실행되도록 프로그래밍하는 것

### if문과 if-else문

- **if문**

	if(조건식){
	
	수행문;
	
	}

- **if-else 문**

	if(조건식){
	
	수행문1;
	
	}else{
	
	수행문2;
	
	}
### if-else if-else문

- if-else-if-else문: 하나의 경우에 조건이 여러 개에 해당하는 경우

   ![Untitled](https://user-images.githubusercontent.com/59382707/130347814-11f3b1af-b1c5-4d0c-9350-c62cd986719c.png)

	if(조건1){
	
	문장1;
	
	}else if(조건2){
	
	문장2;
	
	}else if(조건3){
	
	문장3;
	
	}else{
	
	문장4;
	
	}
